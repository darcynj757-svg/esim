Создай fintech PWA-приложение — аналог сервиса виртуальных карт и финансовых услуг для российских пользователей, оплачивающих зарубежные сервисы. Полноценный MVP со всеми 9 разделами, mock-платежи с архитектурой, готовой под подключение реальных API.

═══════════════════════════════
ТЕХНИЧЕСКИЙ СТЕК
═══════════════════════════════
- Frontend: Next.js 14 (App Router) + TypeScript
- Стили: Tailwind CSS + shadcn/ui
- Бэкенд/БД: Supabase (PostgreSQL + Auth + Row Level Security)
- Состояние: Zustand
- Анимации: framer-motion
- PWA: next-pwa (манифест + service worker + «Добавить на главный экран»)
- Иконки: lucide-react
- Язык интерфейса: русский, mobile-first

═══════════════════════════════
АРХИТЕКТУРА ПЛАТЕЖЕЙ (КЛЮЧЕВОЕ)
═══════════════════════════════
Все платёжные операции — через единый сервисный слой /lib/payments/ с интерфейсом PaymentProvider (методы: issueCard, topup, withdraw — все async, типизированы, статусы pending/success/failed).
- Реализация MockProvider возвращает фейковые данные с задержкой 1-2 сек.
- Структура готова под реальных провайдеров: cardIssuer, cryptoAcquiring (USDT TRC20/ERC20), sbp (СБП).
- В коде комментарии: где и какой реальный API подключать.
- Реальные API подключаются заменой MockProvider без переписывания UI.

═══════════════════════════════
БАЗА ДАННЫХ (Supabase) + seed-скрипт
═══════════════════════════════
Таблицы с включённым RLS (пользователь видит только свои данные):
- users (id, email, telegram_id, created_at)
- cards (id, user_id, type[virtual/plastic], system[visa/mastercard], number_masked, balance, currency, status, created_at)
- transactions (id, user_id, card_id, type[topup/withdraw/payment], method[card/usdt/sbp/cash], amount, currency, status, created_at)
- orders (id, user_id, product_id, product_type, amount, status, created_at)
- gift_cards_catalog (id, name, category, country, denominations[], image, price)
- esim_catalog (id, country, region, data_gb, days, price)
- transfers (id, user_id, from_country, to_country, amount, fee, status)
- reviews (id, user_name, rating, text, created_at)

РАЗДЕЛЫ (все 9)
═══════════════════════════════

1. ГЛАВНАЯ
   - Hero: «Виртуальные и пластиковые зарубежные карты MasterCard и VISA», подзаголовок «Выпуск от 2 минут · Apple Pay / Google Pay · 200 000+ пользователей»
   - CTA: «Выпустить карту» + «Личный кабинет»
   - 4 блока преимуществ: универсальность, безопасность, международное использование, удобное управление
   - Сетка-плитки категорий gift-карт
   - Блок eSIM с популярными странами «от X ₽»
   - FAQ-аккордеон (15+ вопросов: пополнение, вывод, лимиты, Apple/Google Pay, USDT, страны, комиссии)
   - Блок «Что можно оплатить» — сетка логотипов (Netflix, Spotify, Booking, Epic Games, iCloud, ChatGPT, Notion и др.)
   - Карусель отзывов
   - Футер: контакты, телефоны, соцсети, юр-инфо, ссылки на политики

2. КАРТЫ (MasterCard/VISA)
   - Выбор виртуальная/пластиковая, тарифы выпуска, описание (3DS, лимиты, Pay-сервисы)
   - Форма выпуска → mock-выпуск → карта появляется в кабинете

3. GIFT-КАРТЫ
   - Каталог с фильтрами по 30+ категориям (Steam, PlayStation, Xbox, подписки, стриминг, VPN, музыка, такси, еда, крипта и т.д.)
   - Фильтр по стране (флаги), карточка товара → выбор номинала → корзина

4. МАГАЗИН ИГР
   - Каталог игр: платформы (Steam, Xbox, PS, Nintendo), регионы, цены; страница игры с описанием

5. ПОПОЛНЕНИЕ ИГР И СЕРВИСОВ
   - Список сервисов, ввод ID аккаунта, выбор суммы, оформление

6. ДЕНЕЖНЫЕ ПЕРЕВОДЫ
   - Калькулятор: страна-отправитель → страна-получатель (60+ стран, флаги), сумма, авто-расчёт комиссии и итога
   - Направления «Из России» и «В Россию», форма перевода

7. МОБИЛЬНАЯ СВЯЗЬ
   - Пополнение зарубежных операторов: выбор страны/оператора/суммы

8. eSIM
   - Табы: Страны / Регионы / Глобальный; сетка тарифов (GB / дни / цена), карточка покупки

9. ОТЗЫВЫ
   - Лента отзывов с рейтингом-звёздами, форма добавления

ЛИЧНЫЙ КАБИНЕТ:
   - Регистрация/вход через Supabase (email + опционально Telegram)
   - Дашборд: мои карты с балансом, быстрые действия
   - Пополнение: метод (банк.карта / USDT / СБП / наличные) → mock-форма → баланс обновляется
   - Вывод: банк / USDT / СБП
   - История транзакций (фильтры по типу/статусу)
   - Корзина, оформление и история заказов
